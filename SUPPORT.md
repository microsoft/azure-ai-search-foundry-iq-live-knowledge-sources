# Support

This repository contains sample code and accelerator assets. It is not a production service.

For Azure AI Search, Foundry IQ, Fabric IQ, Microsoft Fabric, or MCP product issues, use the appropriate Microsoft support channel for your subscription or tenant.

For repository issues, include:

- The scenario or pattern involved.
- The Azure AI Search API version.
- Redacted request and response payloads.
- Expected behavior.
- Observed behavior.

Do not include secrets, bearer tokens, customer data, or private endpoint details.
